The Workout Warp Wizard

My first project

Inspiration

Our inspiration for WorkoutWarp Wizard came from a shared frustration with mundane workout routines. We sought to create a dynamic tool that injects excitement and variety into fitness regimens.

What it does

WorkoutWarp Wizard is a personalized workout generator. By understanding user preferences, it crafts unique exercise routines, making fitness more engaging and effective.

How we built it

Tech Stack: Leveraging Flask, Python and HTML for a responsive web interface. Algorithm: Developed an algorithm to curate workouts based on user input, ensuring diversity and effectiveness. Collaboration: A collaborative effort, combining individual strengths to create a cohesive and functional application.
